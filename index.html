What's Typescript

Typescript is a typed superset of JavaScript that compiles to plain JavaScript.

Typescript -

  • Adds additional features like Static Type (optional), Class, Module etc. to JavaScript
  • Microsoft technology.
  • Open Source.
  • Versions: First made public in October 2012. Latest version - Typescript 3.2.

Type

TypeScript has next type:

  • Basic Type
    • Number
    • Boolean
    • String
  • And next type:
    • Array
    • Tuple
    • Enum
    • NULL, UNDEFINED, VOID, NEVER
    • ANY

INTERFACE

We can to describe Interface, that serve as templates for objects


interface Block{
	width: number;
	height: number;
}
let first: Block = {width: 10; height: 20};// ok (1)
let second: Block = {width: 10; radius: 3};// Error (2)

Use the optional fields via question mark - '?'.


interface Block{
	width: number;
	height?: number;
	radius?: number;
}
let first: Block = {width: 10; height: 20};// ok (1)
let second: Block = {width: 10; radius: 3};// ok (2)

Readonly properties


interface Block{
	readonly width: number;
	height?: number;
	radius?: number;
}

let field: Block = {width: 10; height: 20}
field.width = 45;// error

Interface can be inherited from each other.


interface Person{
	name: string;
	age: number;
}
interface Employee extends Person{
	title: string;
}
let David: Employee = {
	name: 'David'; // property Person
	age: 25; // property Person
	title: 'Junior JS-developer'
}

CLASSES

Classes in TypeScript are describe in style ES6.


class Greeter {
		greeting: string;
		constructor(message: string) {
		this.greeting = message;
	}
	greet() {
		return "Hello, " + this.greeting;
	}
}

let greeter = new Greeter("world");

Classes are inherited by using the keyword "extends"


class Animal {
	constructor(public name: string) { }
	move(distanceInMeters: number = 0) {
		console.log(`${this.name} 
moved ${distanceInMeters}m.`);
	}
}
class Snake extends Animal {
		constructor(name: string) { super(name); }
		move(distanceInMeters = 5) {
				console.log("Slithering...");
				super.move(distanceInMeters);
		}
}
let sam = new Snake("Sammy the Python");
sam.move();// method Animal

Access modifiers: Public, Private, Protected.

Public fields are full accesible to the user. They are variable directly from the object reference.

Animal.name // value field

Private:

Private fields are hidden from the User. Access to them is only inside the class. Private field don't inheritance.


class Animal{
	private name: string;
	constructor (theName: string){this.name = theName;}
}
let cat: Animal = new Animal('Cat');
cat.name;// error: name is private

Protected field works the same, but it is inheritance.


class Person {
	protected name: string;
	constructor(name: string) { this.name = name; }// Constructor may also be marked protected.
}

class Employee extends Person {
	private department: string;
	constructor(name: string, department: string) {
			super(name);
			this.department = department;
	}

	public getElevatorPitch() {
			return `Hello, my name is ${this.name} and I work in ${this.department}.`;
	}
}

let howard = new Employee("Howard", "Sales");
console.log(howard.getElevatorPitch());
console.log(howard.name); // error

Readonly

You can make properties readonly by using the readonly keyword.


class Octopus {
	readonly name: string;
	readonly numberOfLegs: number = 8;
	constructor (theName: string) {
			this.name = theName;
	}
}
let dad = new Octopus("Man with the 8 strong legs");
dad.name = "Man with the 3-piece suit"; // error! name is readonly.

We can also create static


class Grid {
		static origin = {x: 0, y: 0};
		calculateDistanceFromOrigin(point: {x: number; y: number;}) {
				let xDist = (point.x - Grid.origin.x);
				let yDist = (point.y - Grid.origin.y);
				return Math.sqrt(xDist * xDist + yDist * yDist) / this.scale;
		}
		constructor (public scale: number) { }
}

let grid1 = new Grid(1.0);  // 1x scale
let grid2 = new Grid(5.0);  // 5x scale

console.log(grid1.calculateDistanceFromOrigin({x: 10, y: 10}));
console.log(grid2.calculateDistanceFromOrigin({x: 10, y: 10}));
